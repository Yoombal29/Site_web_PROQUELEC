## Exemples de Requêtes JSON

### Exemple 1 : Trouver les règles pour les salles d'eau
```javascript
json.titre_7.installations_specifiques.salles_eau
// Retourne les articles 701.x avec prescriptions particulières
```

### Exemple 2 : Obtenir les temps de coupure
```javascript
json.titre_4.protection.temps_coupure.tableau_41A
// Retourne les temps max par schéma et tension
```

### Exemple 3 : Extraire les définitions liées à la protection
```javascript
json.definitions.filter(def => def.categorie === "protection_chocs_electriques")
// Retourne 23 définitions sur la protection
```

### Exemple 4 : Lister les lois sénégalaises référencées
```javascript
json.references_legales.senegal
// Retourne codes, lois, décrets et arrêtés
```
