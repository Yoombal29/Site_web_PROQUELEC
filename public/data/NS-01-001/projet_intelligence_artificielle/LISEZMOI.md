# Projet Intelligence Artificielle - NS 01-001 PROQUELEC

Ce dossier contient l'ingénierie documentaire complète de la norme sénégalaise NS 01-001.

## Structure du projet
- **jeux_de_donnees/** : Contient les données nettoyées pour RAG et SFT (formats .json, .jsonl et .txt).
- **code_source/** : Scripts de traitement et pipeline de production.
- **tests_unitaires/** : Rapports d'audit et de validation des données.
- **documentation/** : Rapports finaux et synthèses.

## Note technique
Les fichiers .txt dans 'donnees_entrainement' conservent la structure JSONL pour faciliter le copier-coller dans les interfaces CMS sans altérer le format de données attendu par le moteur Phi 3.5.
